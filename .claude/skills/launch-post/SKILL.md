## 1 — LINKEDIN POST

### OBJECTIVE
- explain clearly  
- show value  
- sound professional but natural  
- slightly storytelling  

### STRUCTURE

**FR:**
- Hook (1–2 lines)  
- Context / problem  
- Solution (product)  
- Key features (short)  
- Closing (optional CTA)  

**EN:**
→ same structure  

### STYLE
- clean  
- not cringe  
- not corporate buzzword heavy  
- slightly personal tone allowed  

### LENGTH
- 8–15 lines max  
- readable on mobile
